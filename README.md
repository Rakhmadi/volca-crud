# volca-boilerplate
boilerplate for Vloca deno framework

# How To Install 
```
> git clone https://github.com/Rakhmadi/volca-boilerplate.git
> cd volca-boilerplate
```

# How To Run
```
> deno run --unstable -A server.ts
```
Using Denon :
```
> denon run --unstable -A server.ts
```
Using Vscode : Press F5
